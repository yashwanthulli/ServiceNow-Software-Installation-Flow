# ServiceNow Software Installation Flow

## Project Overview
This project automates the process of requesting, approving, and installing licensed software using ServiceNow Service Catalog.

## Features
- Service Catalog item for software request
- Approval workflow for manager
- Automatic task creation for IT team
- Email notification for approval
- Task tracking and completion

## Workflow Steps
1. User submits software request
2. Manager approval is triggered
3. Notification is sent to approver
4. Task is created for IT team
5. Software is installed
6. Task is marked as completed
